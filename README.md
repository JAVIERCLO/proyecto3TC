## Descripción

Este proyecto implementa un simulador de Máquinas de Turing (MT) de una cinta que puede:

- Cargar la descripción formal de una MT desde un archivo YAML
- Simular la ejecución de cadenas de entrada
- Mostrar las descripciones instantáneas en cada paso
- Determinar si una cadena es aceptada o rechazada

## Requisitos
- Lenguaje Python


## ¿Clonar el repositorio? 

1. Clonar el repositorio.
```
git clone https://github.com/JAVIERCLO/proyecto3TC.git
```
2. Entrar a la carpeta del repositorio.
```
cd proyecto3TC
```

## Ejecutar los programas
> 1. Para ejecutar la maquina de reconocimiento.

1.1 Ir a la carpeta MT_Reconocedora
```
cd MT_Reconocedora
```
1.2 Ejecutar programa
```
py main.py
```

> 2. Para ejecutar la maquina alternadora.

2.1 Ir a la carpeta MC_Alternadora
```
cd MC_Alternadora
```
2.2 Ejecutar programa
```
py main_alter.py
```
